### Completely eslint and prettier configs to use in your Node and React projects.

## **How to use:**
### Add in your `.eslintrc.json`:

### React
```json
  {
    "extends": ["@pretty-eslint/eslint-config/react"]
  }
```
### Node
```json
  {
    "extends": ["@pretty-eslint/eslint-config/node"]
  }
```
